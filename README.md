# SaarthiX

This is a full-stack career ecosystem platform built with React, Spring Boot, and MongoDB.

## Features

- **Students:** Job Blueprint, Resume Builder, Career Counselling, Apply to Jobs, Courses, Role Ready Training.
- **Institutes:** Internship Management, Student Training (Role Ready), Expert Sessions, Workshops.
- **Industry:** Resume Access, Role Ready Freshers, Post Jobs & Hackathons, AI Technical Interview.

## Tech Stack

- **Frontend:** React, Vite, TailwindCSS, shadcn/ui
- **Backend:** Spring Boot, Gradle
- **Database:** MongoDB
- **Deployment:** Docker, Docker Compose

## Prerequisites

- Docker
- Docker Compose

## Getting Started

1. Clone the repository.
2. Run the following command to start the application:

   ```bash
   docker-compose up --build
   ```

3. The frontend will be available at [http://localhost:3000](http://localhost:3000).
4. The backend will be available at [http://localhost:8080](http://localhost:8080).
